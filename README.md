# 🌍 Explore London

This project is a simple travel webpage about London, created as part of the Code Your Future course.

## 📖 About the Project

The webpage introduces London by highlighting some of its most famous landmarks, attractions, and activities. It includes:

- A short introduction to London
- Images of famous places
- A list of the top things to do
- A modern and responsive layout
- HTML and CSS styling

## 🛠️ Built With

- HTML5
- CSS3

## 🎯 Purpose

The purpose of this project is to practise:

- Writing semantic HTML
- Styling with CSS
- Using images and layouts
- Creating a clean and attractive webpage
